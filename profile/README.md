## About Rust

Rust is a hardcore survival multiplayer sandbox game where players must survive in a harsh open world filled with danger from both the environment and other players. Starting with nothing, players gather resources, craft tools, and build bases to stay alive.

The game emphasizes PvP combat, base building, and resource management. Players can form alliances, raid others, and compete for dominance in a constantly evolving world.

With its realistic survival mechanics, dynamic gameplay, and strong multiplayer focus, Rust delivers an intense experience centered on survival, strategy, and competition.
